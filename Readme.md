# 💻 HealthHub - Dashboard de Monitoramento Hospitalar

A interface do **HealthHub** foi projetada para oferecer clareza máxima e agilidade em ambientes sob pressão, como salas de triagem e postos de enfermagem em hospitais e clínicas de médio porte.

## 🚀 Transformando o Atendimento
Em unidades de saúde, a agilidade salva vidas. O **HealthHub** substitui a prancheta de papel por um dashboard interativo focado em:

### 1. Triagem Ágil e Digital
O processo de triagem deixa de ser um gargalo administrativo. Com formulários otimizados, o profissional de saúde registra o paciente e seus sinais vitais iniciais em segundos, disponibilizando os dados imediatamente para o corpo médico.

### 2. Observação Dinâmica vs. Estática
Diferente de uma ficha de papel, que mostra apenas uma "foto" de um momento isolado, o HealthHub oferece uma **observação dinâmica**. Ao abrir o prontuário digital, o profissional visualiza a última medição e o histórico clínico completo, permitindo identificar tendências perigosas na saúde do paciente de forma precoce.

### 3. Modernização Acessível
O sistema é leve e intuitivo, não exigindo treinamentos complexos ou infraestruturas de TI pesadas. É a solução ideal para democratizar a tecnologia hospitalar em unidades que ainda dependem de processos manuais.

## ✨ Funcionalidades Principais
* **Dashboard Visual:** Cards de pacientes com indicadores de saúde rápidos.
* **Edição em Tempo Real:** Modais organizados por abas (Dados Pessoais, Sinais Hospitalares e Anamnese) para facilitar a leitura.
* **Interface de Alta Performance:** Desenvolvido com **Mantine UI** para garantir navegação fluida e responsiva.

## 🛠️ Tecnologias Utilizadas
* **React + Vite:** Para uma renderização ultra-rápida da interface.
* **TypeScript:** Segurança no manejo de dados e redução de bugs em tempo de desenvolvimento.
* **Axios:** Comunicação eficiente com a API do HealthHub.

## 🔧 Como Iniciar
1. `git clone <url-do-repo>`
2. `npm install`
3. No arquivo `src/services/api.ts`, aponte para o endereço do seu back-end.
4. `npm run dev`
5. Acesse em seu navegador: `http://localhost:5173`.
